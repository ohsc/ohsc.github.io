全面转向WLW离线发布

其实很早以前就开始使用Live Writer，只不过有些功能还不完善，现在借助一些插件，基本上可以满足我所有需求了。

  1. 对WLW有兴趣的朋友可以先去看Digglife写的心得
  2. 解决Wordpress2.3的tag问题，只要把wlwmanifest.xml上传到wp的根目录，更新一下WLW的帐户信息，然后你就能找到“关键字”这个字段。这个就是用来发布tag的。
  3. 代码高亮，我是通过下载两个插件解决的。一个是Highlight4Writer，另外一个是CodeSnippet。因为他们支持的语言有限，所以我用两个相互补充。
  4. 关于插入土豆、优酷等flash格式是视频链接。这个目前我还没有找到适合的方法。一旦要插入这些视频，我必须回到wp，利用coolplayer来实现。如果哪位朋友知道解决方法，请通知我哦。
